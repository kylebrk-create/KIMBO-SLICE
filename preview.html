Image Specs

    Email Best Practices (built-in)

    • ✓ 600px max width layout (widest common denominator)
    • ✓ Bulletproof buttons (VML) for Outlook
    • ✓ Hidden preheader text (85–130 chars)
    • ✓ Live text for headlines/CTAs (don’t bake copy into images)
    • ✓ ALT text on all images; meaningful first frame if using GIF
    • ✓ CAN-SPAM footer with physical address + unsubscribe

    File Types & Sizes

    • Photos: JPG (80–85% quality). Logos/UI: PNG. Avoid WebP (not universal).
    • Hero: ideally **<200 KB**; total email images **<1.2 MB**.
    • Include explicit width on <img> tags; set style="height:auto;".

    Deliverability & SEO-ish signals

    • Use **absolute https URLs** for all images/links.
    • Add **UTM** tags to campaign links (e.g., ?utm_source=email&utm_medium=blast&utm_campaign=elenack-q4).
    • Keep text-to-image balance reasonable (include real copy).
    • ESP level: make sure **SPF, DKIM, DMARC** are passing; use **List-Unsubscribe** header where possible.

    Editing Tips (repeatable process)

    1. Duplicate a template in /templates and rename it (lowercase, hyphens).
    2. Swap the hero URL and copy. Keep image at **600px** wide; compress <200 KB.
    3. Update CTA links + add UTMs. Keep button label to 2–4 words.
    4. Set preheader to 85–130 chars; ensure it complements the subject line.
    5. Update footer address if brand/location changes.
    6. Preview here; then send tests from your ESP to Gmail + Outlook.
